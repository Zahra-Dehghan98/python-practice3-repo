# Python Exercise - BookBuddy Library Manager

A CLI application for book lovers to track reading habits and manage personal library.

## Features:
- Add books with details (title, author, genre, pages)
- Track reading sessions and progress
- Support for different book types (Book, EBook, AudioBook)
- Export/import data in JSON and Pickle formats
- Reading statistics and progress reports

## Technical Features:
- Object-oriented design with inheritance
- Decorators, context managers, custom exceptions
- File handling and data persistence
- Error handling and logging

## How to run:
```bash
python main.py
